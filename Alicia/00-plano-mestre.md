# AlicIA - Plano Mestre

## Resumo executivo
AlicIA será um terminal open source, focado em alta performance, com agentes de IA e UX amigável para pessoas sem prática em terminal.

Decisões já fechadas:
- Nome: AlicIA
- Licença: MIT
- UI: egui (nativo)
- SO alvo: Windows + macOS + Linux com paridade desde o início
- Segurança: bloquear acesso fora do workspace no MVP
- Política de rede: incluída no MVP
- Persistência no MVP: logs em arquivo

## Objetivos do produto
1. Entregar performance próxima de terminais de referência.
2. Garantir um fluxo de aprovação simples para ações sensíveis.
3. Permitir execução de agentes em terminal real com controle de risco.
4. Tornar tarefas comuns acessíveis para iniciantes.

## Não objetivos iniciais
1. Reimplementação de OAuth de terceiros.
2. Marketplace complexo de plugins.
3. Colaboração em tempo real multiusuário.
4. Execução distribuída remota de agentes.

## Entregáveis da fase inicial
1. Documento de arquitetura técnica.
2. Documento de modelo de segurança e permissões.
3. Documento de roadmap por marcos.
4. Documento de UX para iniciantes.
5. Documento de riscos e mitigação.
6. Backlog MVP priorizado.

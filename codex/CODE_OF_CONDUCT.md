# Code of Conduct

This repository expects respectful interactions in issues, pull requests, discussions, and reviews.

## Expected Behavior

- Be respectful and inclusive.
- Focus on objective technical feedback, not personal attacks.
- Do not post harassment, discrimination, hate speech, spam, or doxxing.
- Do not share secrets, personal data, or malicious/illegal content.

## Enforcement

Violations may result in content moderation, discussion closure, or restriction of repository access.

## Reporting

1. For public cases, open an issue with the prefix `[CoC]`.
2. For sensitive cases, contact repository maintainers directly via GitHub.

# Contributing

For the full contribution policy, see `docs/contributing.md`.

## Invitation-Only Code Contributions

1. External code contributions are accepted by maintainer invitation only.
2. Before opening a pull request, open or comment on an issue to align on scope and approach.
3. Pull requests submitted without explicit invitation may be closed without review.
4. Non-code contributions (reproduction details, analysis, and feedback) are welcome in issues.
